Project 404

Project 404 explores the dualism of mobility as it pertains to the physical and virtual realms. This dualism examines the attributes of form and function, drawing evidence that devices and active systems reflect most antiquated communication methods such as cave paintings and hieroglyphs. The name 404, inspired by a code in response to a client request, questions the overall state of existence in pertinence to physical perception and mental projection via mobility.

Process & Ideation

Overview

Error code 404 alerts the user client that a page does not exist on the internet. The internet is an ever expanding platform that contains several different domains, all connected via hyperlinks. Whatever a user client inputs in the address bar has a final destination, and if the page does not exist wether or not the html document exist. This concept goes into the idea of mobility, being that it involves a starting and ending point.

Typeface

I created a code inspired typeface which binds both the formal traits of glyphs and architectural structure of a maze. In making the first iteration of the typeface, 404 Regular, I surrounded each glyph with a common frame. Furthermore, I started creating several different paths, reiterating the surrounding structures of each letter. This design execution functions to bridge both sides of the frames connecting each glyph.

3D Rendering

I rendered the maze in black and white in order to capture the dark space of the web. This unsettling atmosphere, void of color, is what I imagine the internet to be. The inner parts of the walls, obscured from the viewer's perspective, emits light connecting each path of the maze, similar to hyperlinks. The viewer is able to experience the maze externally and internally.

Final Deliverables

In exploration of constructing this typeface, I experimented with its utilization. The typeface in its formal and symbolic quality is capable of projecting both words, via character, and paths by design. I created different styles and levels of contrast to each letter. I randomized each letter by character assignment, via the keyboard. Users are able to focus more on the maze as they are able to type in new paths with each characters.
